# 🇰🇷 국내 레버리지 ETF 알람 시스템 v2.0

네이버 금융 실시간 데이터 기반
SMA120 + ATR(20)×2.0 트레일링스탑 + SMA20 재진입 전략
브라우저 팝업 알람 + 실시간 수익률 계산

## 📈 매매 전략

```
매수:    SMA120 위 + gap 0~15% → 즉시 전액 매수
ATR매도: gap>20% + ATR(20)×2.0 트레일링스탑 터치 → 즉시 전액 매도
SMA매도: SMA120 이탈 → 즉시 전액 매도
재진입:  ATR매도 후 SMA20 재돌파 + SMA120 위 → 재매수
대기:    현금 보유
```

## 📱 화면 구성 (3탭)

### Tab 1 - 대시보드
- 4개 시장 실시간 신호 카드
- 현재가 / SMA120 / gap% / SMA20 / ATR스탑
- 사전경보 D-3 / D-2 / D-1
- 1분마다 자동 갱신
- 신호 발생 시 브라우저 팝업 알람

### Tab 2 - 투자현황
- 시장별 투자금 / 매수가 / 보유수량
- 시장별 실시간 수익금 / 수익률
- 전체 합산 수익금 / 수익률

### Tab 3 - 설정
- 팝업 알람 (끄기 / 소리+진동 / 무음)
- 시장별 투자금액 + 평균매수가 입력
- 키움증권 API (추후 자동매매 연동)

## 🔔 알람 체계

| 신호 | 조건 |
|------|------|
| 🔴 매수 신호 | SMA120 위 + gap 0~15% |
| 🔴 재진입 신호 | ATR매도 후 SMA20 재돌파 |
| 🔵 ATR 매도 | gap>20% + 트레일링스탑 터치 |
| 🔵 SMA 매도 | SMA120 이탈 |
| 📌 D-3 경보 | 매수 gap 22~26% / 매도 스탑 5~10% |
| ⚠️ D-2 경보 | 매수 gap 18~22% / 매도 스탑 2~5% |
| 🚨 D-1 경보 | 매수 gap 15~18% / 매도 스탑 0~2% |

## 💰 설정 저장 방식

